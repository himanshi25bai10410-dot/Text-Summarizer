# 🧠 Text Summarizer

## 📌 Description
This project is a simple AI-based Text Summarizer built using Python and Natural Language Processing (NLP). It takes long text as input and generates short, meaningful summaries by extracting the most important sentences. This tool helps users quickly understand the main idea without reading the entire content.

---

## 🎯 Objective
The goal of this project is to reduce reading time and make it easier to grasp key information by automatically summarizing long text into concise and relevant content.

---

## 🚀 Features
- Accepts long text as input  
- Generates concise summaries  
- Uses NLP techniques for processing  
- Easy-to-use command-line interface  
- Interactive browser-based UI using Streamlit  
- Saves time and improves readability  

---

## 🛠️ Technologies Used
- Python  
- Natural Language Processing (NLP)  
- NLTK library  
- Streamlit

## 📦 Requirements
Make sure you have the following installed on your system:

- Python 3.x  
- pip (Python package manager)  

Required Python libraries:  

```bash
nltk
streamlit

⚙️ Installation
1.Clone the repository:
git clone https://github.com/your-username/Text-Summarizer.git
2.Navigate to the project folder:
cd Text-Summarizer
3.Install required Python libraries:
pip install -r requirements.txt
If requirements.txt does not exist, install manually:
pip install nltk streamlit

▶️ Usage
Option 1: Command-line
Run the program in terminal:
python main.py

Option 2: Streamlit UI (Recommended)
Run the UI in browser:
streamlit run app.py
-Enter your text in the text area
-Use the slider to select number of sentences in the summary
-Click Summarize to see the output
